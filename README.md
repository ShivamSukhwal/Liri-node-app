# Liri-node-app

Search Spotify for songs, Bands in Town for concerts, and OMDB for movies
Liri is an app that runs off javascript and Node.js.
With the help of Liri you can search for concert dates, song info , as well as movie information

to use liri all you have to do is

1. open up the command line
2. and travel to wherever you have placed liri ex. cd desktop cd Liri-node-app
3. once your in the right repository on your command line type node liri.js along with the command as well as the item you are searching for
4. If your looking up a concert date you would type in node liri.js concert lady-gaga
5. If your looking up movie information you would type node liri.js movie The Hunger-games
6. If your looking up a song title you would type node liri.js song we-are-the-champions
7. avoid spaces and use "-" if neccessary
